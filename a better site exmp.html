What to Study in This Code

1. CSS Reset: The * selector at the top resets default browser styles for consistency.

2. CSS Variables: The :root selector defines reusable values (colors, spacing, etc.)

3. Flexbox & Grid: Used for layouts (navigation, projects grid, contact items)

4. Responsive Design: Media queries adjust the layout for different screen sizes

5. CSS Positioning: Fixed header, absolute positioning for the mobile menu

6. Pseudo-elements: ::after used for underline effects and decorative elements

7. Transitions & Transforms: Used for hover effects and animations

8. Semantic HTML: Proper use of section, header, footer, nav tags
